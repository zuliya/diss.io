Welcome to the Free Forex API !!

This API is available free of charge to all web developers who need to show the most accurate, live, foreign exchange rates for all the major currency pairs. The API is simple and easy to use and best of all there is no registration required! Simply make a call to the end point as described in the API Documentation. All that we ask is that you kindly add a backlink to our site to support our project and allow us to continue to provide this great service for free!
